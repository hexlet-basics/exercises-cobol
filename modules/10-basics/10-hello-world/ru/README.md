Изучать язык программирования, по традиции, начинают с программы 'Hello, World!'.

```text
Hello, World!
```

В языке COBOL эта программа будет выглядеть так:

```cobol
IDENTIFICATION DIVISION.
PROGRAM-ID. Index.
PROCEDURE DIVISION.
DISPLAY "Hello, World!".
STOP RUN.
```

Текст `Hello, World!` появится на экране благодаря команде `DISPLAY`.
